Welcome to a timetable management system for colleges and universities to schedule their organization's timetable.Schools, Colleges, and Universities can use this application inorder to create and manage timetable and help them to generate their timetable with ease and maximum efficiency, thus saving the organization's valuable time.

Get Started »   

1. Add Professor

Navigate to the "Add Professor" section and add the corresponding details. Incorrect details can also be removed via the "Delete/Update" option.

2. Add ClassRooms

Navigate to the "Add Rooms" section and add the corresponding details. Incorrect details can also be removed via the "Delete/Update" option.

3. Add Courses

Navigate to the "Add Courses" section and add the corresponding details. Incorrect details can also be removed via the "Delete/Update" option.


4. Generate Timetable

Navigate to the "Generate Timetable" section and click on "MSC/MCA" option.Multiple combination of timetable gets generated for respective program out which one get implemented in orgranisation depending organisation decision.
